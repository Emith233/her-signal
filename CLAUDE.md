# 城市探索 · 监控解谜游戏

## 项目概览

一款**黑客题材的城市信号源探索解谜游戏**。玩家扮演一名身份未知的黑客，受委托人 F 的委托寻找失踪的女孩「小初」。通过接入城市信号源网络，解锁节点、分析信息碎片（图片、帖子、记录等），一步步逼近真相。

## 当前进度

- 第一版 demo 已搭建：开场终端动画 + 主界面（信号源拓扑图 + 小E对话 + 控制台）
- 22 个信号节点分布在城市各区，初始仅 #07 亮起
- 基础指令系统已就绪

## 技术栈

- 纯前端单文件 HTML（`index.html`）
- 无框架、无构建工具
- 内联 CSS + JavaScript
- 约 883 行

## 游戏设定

- **主角**：身份未知的黑客
- **委托人**：F（匿名加密渠道）
- **失踪者**：小初（名字取自愚者牌——0号牌，「初」即开端）
- **助手**：小E（AI 助手，协助分析监控成像）
- **机制**：通过解谜点亮监控节点，定位小初的位置
- **塔罗对应**：22 个节点对应 22 张大阿尔卡纳，最后节点为正位愚者
- **塔罗对玩家隐藏**：游戏中不直接透露塔罗牌信息，只在幕后设定中存在

## 当前操作方式

- **点击相邻节点** → 尝试接入/切换过去
- **`scan` / `扫描`** → 查看当前节点监控画面
- **`nodes` / `节点`** → 查看已解锁的节点列表
- **`help` / `帮助`** → 查看帮助
- **`clear` / `清除`** → 清屏
- **输入文字** → 和小E对话

## 监控网络

22 个节点分属不同城区（北城、东城、南城、西城、中心区、外环），通过线路连接。初始仅起始节点亮起，需要沿着线路一步步点击相邻节点接入。

节点在地图上的位置已打乱过一次（混合分布），不再变动。
